DICTIONARIES

  • Dictionary is an unordersd collection of items
  • while other compound datatypes have only value as on element

Create dictionary

A dictionary is as simple as placingitems inside curly braces{},seperated by commas.
An item has a key and the corresponding value expressed as a pair,key:value.
keys must be immutable type and must be unique.

Example:1

  1. # empty dictionary
  2. my_dictionary = {}
  3. print(my_dict)
  4. output: {}

Example:2

  1. # dictionary with integer keys
  2. my_dict = {1:"apple",2:"ball"}
  3. print(my_dict)
  4. output:{1:'apple',2:'ball'}

Example:3

  1. #dictionary with mixed keys
  2. my_dict = {'name':'john', 1:[2,4,3]}
  3. print(my_dict)
  4. output:{'name':'john', 1:[2,4,3]}

Access dictionary elements

indexing is used other container types to access values,dictionary uses keys.

key can be used either inside square brackets or with the get() method.

example:

  1. my_dict = {'name':'jack', 'age':26}
  2. print(my_dict)
  3. output: jack
  4. print(my_dict.get('age')]
  5. output: 26

    6. change or add elements in dictionary

    dictionaries are mutable. we can add new items or change the value of existing items using assignment operator

    if key is already present value gets updated,else a new key:value pair is added to the dictionaries

    Example

    1. my_dict = {'name':'jack', 'age':26}
    2. #update value
    3. my_dict['age'] = 27
    4. print(my_dict)
    5. output: {'age':27,'name':'jack'}
    6. #add item
    7. my_dict['address'] = 'warangal'
    8. print(my_dict)
    9. output: {'address':'warangal', 'age':27, 'name':'jack'}
