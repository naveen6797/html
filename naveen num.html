Number data type in python

python supports integers,floating point numbers and complex numbers.they are defined as int,float and complex.

integers and floating points are seperated by the presence or obsence of a decimal point 8 is integer where as 8.0 is floating point number.

complex numbers are written in the form i.e; x+ij, where x is the real part and y is the imaginary part.

we can use the type() function to know which class a variable.

EXAMPLE:

a = 5

print(type(a))

#ouput:class'int'

print(type(5.0)

#output:class'float'

c=5+3j

print(c+3)

#output:(8+3j)

Numbers we deal witheveryday are decimal(base10)number system.but computer program need to work with binary (base 2),octal (base 8), and hexadecimal(base 16).

in python we can represent there numbers by oppropriatley placing a prefix before that number.

number system prefix for python numbers

number system prefix
binary '0b' or '0B'
octal '0o or '0o'
Hexadecimal '0x or 'oX 
binary-> 0,1
octal-> 0,1,2,...,7.
hexadecimal-> 0,1,2,....9,a,b,...,f.

EXAMPLE

print(0b1101011)

#output:107

print(0o15)

#output:13

print(0xFB)

#output:251
