INPUT:input()

  • The input() function allows user input.

input function first takes the input from the user and then evaluates the expression, which means Python automatically identifies whether user entered a string or a number or list. If the input provided is not correct then either syntax error or exception is raised by python.

Example


num = input("enter number")
print(num)

#output:
enter number:123
123

input function works in python:

  • When input() function executes program flow will be stopped until the user has given an input.
  • Whatever you enter as input, input function convert it into a string. if you enter an integer value still input() function convert it into a string. You need to explicitly convert it into an integer in your code using typecasting.

OUTPUT:print()

The simplest way to produce output is using the print() function where you can pass zero or more expressions separated by commas.

Syntax:

  • print(value(s), sep= ‘ ‘, end = ‘\n’, file=file, flush=flush)

    • Parameters:

    Values:

    • Any value, and as many as you like. Will be converted to string before printed

    sep='separator':

    • (Optional) Specify how to separate the objects, if there is more than one.Default :’ ‘

    end='end':

    • (Optional) Specify what to print at the end.Default : ‘\n’

    file:

    • (Optional) An object with a write method. Default :sys.stdout

    flush:

    • (Optional) A Boolean, specifying if the output is flushed (True) or buffered (False). Default: False

    Example:

    print("naveen mateti")
#output:naveen mateti

print("sai pavan", end="*")
#output:sai pavan*

print("my name is", "naveen mateti")
#output:my name is naveen mateti

print("my name is ", "naveen", sep="*")
#output:my name is *naveen

    Output formatting:
    

    There are several ways to present the output of a program, data can be printed in a human-readable form, or written to a file for future use. Sometimes user often wants more control the formatting of output than simply printing space-separated values. There are several ways to format output.
    

      

    • To use formatted string literals, begin a string with f or F before the opening quotation mark or triple quotation mark.
      • 

    • The str.format() method of strings help a user to get a fancier Output
      • 

      Example:
      

      print('my name is {}.'.format("naveen"))
#output:my name is naveen.

n1 = "naveen"
n2 = "pavan"
print('we are {} and {}.'.format(n1, n2)
#output:we are naveen and pavan.

n1 = "naveen"
n2 = "pavan"
print('we are {1} and {0}.'.format(n1,n2))
#output:we are pavan and naveen.

print('we are {n3}, {n2} and {n1}.'.format(n1="naveen", n2="pavan", n3="raju"))
#output:we are raju, pavan and naveen.
