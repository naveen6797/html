LIST

  • list is one of data type in python
  • A list in python is mutable, or ardered sequence of elements
  • list is one of the most frequently used and very versatile datatype used in python

Create list

A list is created by placing all the iteams inside asquare bracket[],seperated by commas.
A list can have any number of items and they may be of different type (integer, float, string,etc)

Example:1

  1. # empty list
  2. my_list = []
  3. print(my_list)
  4. output: []

Example:2

  1. # list having integers
  2. my_list = [1,2,3]
  3. print(my_list)
  4. output:[1,2,3]

Example:3

  1. #list with mixed data types
  2. my_list = [1,'hello',3.4]
  3. output:[1,'hello',3.4]

Access list elements

There are various ways in which we can access the elements of a list.

1. Indexing

  • we can use the index operator[]to access on item in a list where the index start from 0
  • the index must be an integer; so we cannot use float or othertypes,that will be result into type error

    • Example

    1. my_list = ['p','r','o','b']
    2. print(my_list[0])
    3. output: p

    2.Negative indexing

    The index of -1 refers to the last item,-2 to the second last item and so on

    Example

    1. my_list = ['p','r','o','b']
    2. print(my_list[-1])
    3. output: b

    3.slicing

    we can access a range of items in a list by using the slicing operator-colon ":"

    1. my_list = ['p','r','o','b','e']
    2. print(my_list[1:4])
    3. output: o,b,e
