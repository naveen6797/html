Modules

  • module is a file consisting of python code and it's also runnable code.it can define function ,classes and variables.

Create a module

  • save the code file extension called .py

example

def function(name):
     print("hello"+name)
  • save this code file name call pavan.py

    Import statement

    • we can use module by using import statement

    example

    • Import the module named pavan, and call the function:
    import pavan
     pavan.function("bro")
output:
hello bro
    When using a function from a module, use the syntax: module_name.function_name.

    Re-naming a Module

    • You can create an alias when you import a module, by using the as keyword:

    example

    • Create an alias for pavan called rsp:
    • import pavan as rsp
      rsp.function("world")
output:
hello world

    dir( ) Function

    The dir() built-in function returns a sorted list of strings containing the names defined by a module.

    The list contains the names of all the modules, variables and functions that are defined in a module

    import pavan as rsp
rsp.function("world")
x=dir(rsp)
print(x)
output:
['__builtins__', '__cached__', '__doc__', '__file__', '__loader__', '__name__', '__package__', '__spec__', 'function']
