Conditinal statements

There will be various situations while writing a program when you will have to take care of different possible conditions that might arise while execution of the program. In such situations, if conditions can be used.

  • if statement
  • else statement
  • elif statements
  • nested if statements
  • switch case statements

What is if statement ?

In Python, If Statement is used for decision making. It will run the body of code only when IF statement is true.

When you want to justify one condition while the other condition is not true, then you use "if statement".

syntax:

  1. if condition
  2. statement

Example

  1. a=2
  2. b=3
  3. The if Statement checks for condition a less then b which is True in this case
  4. if (a < b):
  5. print( a less then b)
  6. output: a less then b

what is else statement ?

  • An else statement can be combined with an if statement. An else statement contains the block of code that executes if the conditional expression in the if statement resolves to 0 or a FALSE value.
  • The else statement is an optional statement and there could be at most only one else statement following if.

syntax:

  1. if condition
  2. statement
  3. else condition
  4. satement

Example

  1. a=2
  2. b=3
  3. The if Statement checks for condition a less then b which is True in this case
  4. if (a < b):
  5. print( a less then b)
  6. else
  7. print(a greater then b)
  8. output: a greater then b

what is else if condition

  • To correct the previous error made by "else condition", we can use "elif" statement. By using "elif" condition, you are telling the program to print out the third condition or possibility when the other condition goes wrong or incorrect.

Example

  1. a=2
  2. b=2
  3. The if Statement checks for condition a less then b which is True in this case
  4. if (a < b):
  5. print( a less then b)
  6. elif ( a == b ):
  7. print(a is same as b)
  8. else
  9. print(a greater then b)
  10. output: a is same as b
